ТИП СЕЛЕКТОРАСИНТАКСИС СЕЛЕКТОРАВЕС СЕЛЕКТОРА
Селекторы типа выбирают элементы HTML-документа по их тегу.
Например, селектор p выберет все тегиpна странице.
Селекторы класса определяют стиль элементов с определённым классом.
Селекторы идентификатора выбирают элемент по его уникальному идентификатору.
Селекторы наличия и значения атрибута находят все элементы, которые имеют определённый атрибут или значение атрибута. Например, здесь мы выбираем все ссылки, которые начинаются наhttp:// , но не имеют example.com.
Универсальный селектор позволяет прописать стили для всех элементов HTML без исключения. Он указывается в виде звёздочки *. В примере ниже мы используем универсальный селектор с псевдоэлементами, чтобы изменить расчёт общего размера элемента.

Пример селектора типа:

      p {
        font-size: 16px;
      }

Пример селектора класса:

      .highlight {
        background-color: yellow;
      }

Пример селектора наличия:

      a[href^="http://"]:not([href*="[example.com](http://example.com/)"]) {
        color: #000;
        text-decoration: none;
      }

Пример селектора универсального

      *,
      *::before,
      *::after {
        box-sizing: border-box;
      }

Селекторы-комбинаторы

Комбинаторы выбирают элементы, основываясь на их взаимосвязи в дереве DOM. Можно выбрать элементы, которые являются потомками, соседями или родителями других элементов.

Створимо сторінку, на якій буде описано, якими бувають селектори. Пишемо просто приклад селектора та опис у кілька слів, які елементи будуть обрані на сторінці. > Друга частина завдання: Потрібно створити 10 будь-яких елементів із текстом на сторінці та кожному з них призначити різний колір за допомогою різних селекторів. У нас є як мінімум 5 типів селекторів: за назвою елемента за id за класом за кількома класами за вкладеністю Ці 5 типів обов'язково повинні бути на сторінці, решта — на свій розсуд.
